# Sanad Website

A modern, responsive portfolio website designed to provide a clean, interactive, and user-focused experience across desktop, tablet, and mobile devices.

## Overview

This project showcases personal and professional information through a modern interface with an emphasis on performance, responsiveness, accessibility, and maintainable code. The website is built to deliver a seamless browsing experience while presenting content in a clear and engaging way.

## Features

* Responsive design
* Modern and intuitive user interface
* Optimized performance
* Cross-browser compatibility
* Mobile-friendly layout
* Clean project structure
* Scalable and maintainable codebase
